# CodeAlpha Backend Tasks

This repository contains my CodeAlpha backend development internship tasks built using Express.js and SQLite.

## Included Tasks

### Task 1: Simple URL Shortener
A backend project that accepts long URLs, generates short codes, stores them in SQLite, and redirects users to the original URL.

### Task 2: Event Registration System
A backend project that manages events, users, and registrations using Express.js and SQLite. It supports event creation, user creation, event registration, viewing registrations, and cancellation.

## Tech Stack
- Node.js
- Express.js
- SQLite
